03 UI Component LogicComponent 1 — The Aaj Tak-Style Breaking News TickerHow it works (technical): Backend: NestJS WebSocket gateway subscribes to Redis channel breaking:new. When admin saves a "Breaking" post, a Redis PUBLISH fires. Gateway broadcasts to all connected Next.js frontend clients via socket.io. React updates ticker state without page reload. Latency target: < 2 seconds from Admin Save to user screen.Ticker Logic Rules:Max 15 items in the ticker at any time. 16th pushes oldest out.Items auto-expire after 72 hours (configurable in Admin).Ticker text is pulled from article_translations.title for the active language — fully multilingual.Click on any ticker item → opens full article in new tab.Admin can manually "pin" an item to keep it in ticker indefinitely.Animation speed: 60px/second (CSS animation). Pauses on hover.Component 2 — Automated Hero LayoutHero Auto-Population Logic (Next.js getStaticProps): Queries PostgreSQL: WHERE priority='top_story' AND status='published' ORDER BY published_at DESC LIMIT 1. Featured sidebar: WHERE priority='featured' LIMIT 3. If fewer than 3 featured exist, fills remaining slots from priority='normal' ordered by view_count DESC. Zero manual intervention.Component 3 — NK News-Style Deep Analysis BlockFor long-form investigative content, the system auto-detects articles tagged "Analysis" or "Investigation" and renders them with a special deep-reading layout including counters for Primary Sources, Govt Documents, and Expert Quotes.Video Gallery Component (Auto-Updates): When admin adds a post with content_type='video', it automatically appears in the Video Gallery section. Gallery is a React component that queries /api/videos?limit=12&lang={lang} — an ISR-cached Next.js API route.
