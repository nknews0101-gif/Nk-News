Deliverable 01

System Flow: Admin Click → Frontend Display

The Automated Layout Engine is the heart of NK News. Once an admin assigns a Priority, zero manual design work is required. The system routes, renders, and distributes the content automatically.

Phase A — Content Ingestion

1
Admin Opens Universal Editor Login → CMS Dashboard → "New Post" → Selects Content Type (Article / Video / Photo Story / Live)
2
Content Input & Media Upload Admin writes in any Indian language. Media (photo/video/PDF) uploaded → stored in S3 bucket. YouTube URL pasted → thumbnail auto-fetched via YouTube API.
3
Auto-Translation Trigger (Bhashini) Admin clicks "Auto-Translate to All Languages" → NestJS calls Bhashini ULCA API → Translations stored as language_content rows in PostgreSQL. Admin can edit each translation before publish.
4
Priority & Category Assignment Admin selects: Priority [Breaking / Top Story / Featured / Normal] and Category [Politics / Tech / Sports / Business / Culture / …]. Tags added. Expiry date set (optional).
5
Submit → Backend Validation Pipeline NestJS validation layer checks: slug uniqueness, required fields, media dimensions, profanity filter (optional), IT Rules 2021 disclosure flag.

Phase B — Media Processing & Storage

📷

Image Pipeline

Sharp.js auto-converts uploads → WebP. Generates 4 responsive sizes (320w, 640w, 960w, 1280w). Stores in S3 with CloudFront CDN. URL saved to DB.

▶️

Video Pipeline

YouTube URL → YouTube Data API v3 fetches thumbnail, title, duration. For uploaded videos → MediaConvert generates HLS streams. Thumbnail stored in S3.

🔴

Breaking News Pipeline

Priority = Breaking triggers Redis pub/sub → Next.js frontend subscribes via WebSocket → Ticker updates within 2 seconds without page reload. Redis TTL = 72 hours.

🌐

SEO Pipeline

Slug auto-generated (transliterated for Indian languages). Sitemap.xml regenerated. hreflang tags injected for all 22 language variants. OpenGraph metadata written.

Phase C — Automated Layout Routing Engine

This is the "hands-off" engine. Based solely on Priority, the system positions content automatically:

Phase D — Frontend Delivery

6
Next.js ISR (Incremental Static Regeneration) Homepage regenerates every 60 seconds. Category pages every 120 seconds. Article pages on-demand (revalidate on publish). Served from CloudFront edge.
7
Language Detection & Routing User's Accept-Language header parsed. Cookie preference checked. URL prefix enforced (e.g. /hi/, /ta/, /bn/). Next.js i18n middleware handles redirection.
8
Ad Injection (Automated) AdSense slots in predefined positions. Direct ads: system checks start/expiry dates. Every 3rd paragraph → ad slot injected via React component. CTR data fed back to Ad Manager.
9
Frontend Renders — Zero Admin Design Work The layout is fully automated. Admin never touches the frontend. Ticker, hero, grid, sidebar, video gallery — all populated dynamically from DB + Redis.

Deliverable 03

UI Component Logic

Two signature patterns define the NK News frontend: the Aaj Tak-style live ticker for speed, and the NK News-style deep-analysis block for credibility. Both are automated — zero admin design input required.

Component 1 — The Aaj Tak-Style Breaking News Ticker

🔴 BREAKING
PM Modi inaugurates new expressway in UP | India registers 3.2% GDP growth in Q3 | Supreme Court upholds electoral bond ruling | Delhi air quality index drops to 180 | ISRO announces Chandrayaan-4 launch date | Sensex crosses 85,000 mark for first time
How it works (technical) Backend: NestJS WebSocket gateway subscribes to Redis channel breaking:new. When admin saves a "Breaking" post, a Redis PUBLISH fires. Gateway broadcasts to all connected Next.js frontend clients via socket.io. React updates ticker state without page reload. Latency target: <2 seconds from Admin Save to user screen.

Ticker Logic Rules

  • Max 15 items in the ticker at any time. 16th pushes oldest out.
  • Items auto-expire after 72 hours (configurable in Admin).
  • Ticker text is pulled from article_translations.title for the active language — fully multilingual.
  • Click on any ticker item → opens full article in new tab.
  • Admin can manually "pin" an item to keep it in ticker indefinitely.
  • Animation speed: 60px/second (CSS animation). Pauses on hover.

Video Gallery Component (Auto-Updates)

When admin adds a post with content_type='video', it automatically appears in the Video Gallery section. Gallery is a React component that queries /api/videos?limit=12&lang={lang} — an ISR-cached Next.js API route. YouTube embeds are lazy-loaded with Intersection Observer. Live stream: admin pastes YouTube Live URL → system detects isLive=true via YouTube API → renders as live player with red "LIVE" badge.

Bhashini API Integration

1
Register on Bhashini ULCA Portal

Obtain API key from bhashini.gov.in. Register NK News as a media organization for higher rate limits (recommended: 10,000 calls/day free tier).

2
Pipeline Configuration API Call

POST to https://meity-auth.ulcacontrib.org/ulca/apis/v0/model/getModelsPipeline with source language, target language, and task type (translation). Get pipeline task sequence ID.

3
Translation Inference Call

POST to inference endpoint with text payload. Bhashini routes to best available model (e.g. IndicTrans2 for Hindi, AI4Bharat models for other languages). Response contains translated text in target script.

4
Parallel Batch Processing

NestJS BullMQ job queue processes all 22 language translations in parallel (not sequential). Estimated total time: 30–60 seconds for a 1000-word article. Results saved to article_translations table.

5
Fallback: Google Translate API

If Bhashini returns error or is rate-limited, NestJS falls back to Google Translate API v3. Fallback translations are flagged as translation_source='google' in the DB for editorial review.

Unicode & Font Strategy

Devanagari Script Group

Hindi, Marathi, Sanskrit, Nepali, Dogri, Konkani, Maithili → Noto Sans Devanagari (Google Fonts, preloaded). Line height: 1.8 for readability.

South Indian Scripts

Tamil → Noto Sans Tamil. Telugu → Noto Sans Telugu. Kannada → Noto Sans Kannada. Malayalam → Noto Sans Malayalam. All loaded via font-display: swap.

East Indian & Others

Bengali/Assamese → Noto Sans Bengali. Odia → Noto Sans Odia. Punjabi → Noto Sans Gurmukhi. Urdu → Noto Nastaliq Urdu (RTL support enabled).

RTL Support (Urdu/Kashmiri)

Next.js dir="rtl" set when lang='ur' or lang='ks'. Tailwind CSS RTL plugin used. Flexbox direction auto-reverses. Ticker scrolls right-to-left.

Language Switcher UX

Sticky language selector in the header shows the current language name in its own script (e.g. "हिंदी" not "Hindi"). Dropdown shows all 22 languages grouped by script family. Preference stored in cookie (nk_lang) and localStorage. Next.js middleware reads cookie on every request and redirects to correct /[lang]/ prefix.

Legal & Regulatory

IT Rules 2021 & MIB Compliance

Grievance Officer (Rule 2(w))

Dedicated /grievance page with Officer name, email, physical address, phone number. Complaint form with auto-ticketing (24hr acknowledgement, 15-day resolution mandate). Database table: grievances.

MIB Publisher Disclosure

Footer contains: Owner name, registration number, editor name, registered address, state, country. Visible on every page. Pulled from a site_settings table — no hardcoding.

DPDP Act 2023

Cookie consent banner (first visit). Privacy policy page. Data minimization: we store only name + email for accounts. No behavioral tracking sold to third parties. Plausible Analytics (cookieless).

Content Takedown (Rule 3)

Admin can unpublish any article in 1 click. Automated takedown if flagged by MIB order: status='taken_down' with order reference ID stored. Returns 451 (Legal Reasons) HTTP status.

Critical: Monthly Compliance Report The Admin panel includes a "Compliance Dashboard" that auto-generates the monthly report required under IT Rules 2021 Rule 4(1)(d): number of complaints received, resolved, pending, and content removed. Exportable as PDF for MIB submission.
